F1 - меню выбора модели/операций
Ctrl + Backspace = F1
F2 - настройки портов
F4 - звук показывает/скрывает
F5 - открывает файл config.ini
F6 - открывает файл модели (например Horizont 42LE55D.ini)

Enter - запуск полного тестирования
стрелочки вверх/вниз - выбор операций
ctrl + Enter = запуск выбранной (одной) операции
Двойной клик мыши по операции = ctrl + Enter


Настройка ОТОБРАЖАЕМЫХ операций и моделей в меню выбора модели/операции:


файл config.ini:
пункты раздела [models]     = список отображаемых моделей
пункты раздела [operations] = список отображаемых операций


Настройка ЗАГРУЖАЕМЫХ операций:


Пункты раздела [operations] из файла config.ini  должны 
совпадать с названиями разделов [*раздел*] в файле модели (*имя_модели*.ini)

Соответсвенно имя файла модели (*имя_модели*.ini) должно совпадать с одним из пунктов раздела [models] из файла config.ini


раздел [все команды] - пример всех команд для модели
